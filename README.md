# UC-SemiSeg
本文提出一种基于不确定性校准的教师—学生半监督分割框架（UC-SemiSeg）。该方法通过引入蒙特卡洛随机采样（Monte Carlo Dropout, MCD）对模型预测进行多次随机扰动，从而构建像素级不确定性度量。在此基础上，设计软可靠性过滤（Soft Reliability Filtering, SRF）机制，将预测方差映射为连续权重，实现对伪标签的自适应筛选，在抑制高不确定性噪声的同时保留潜在有效信息。进一步地，在一致性学习中引入不确定性感知加权策略，以提升模型利用无标签数据时的稳定性。同时，采用Attention U-Net作为基础分割网络，以增强对复杂病变区域及边界结构的表征能力。
